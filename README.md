# SALES REGISTRATION  PROJECT

## Description
This project is a basic python application that allows to the user to register product names, prices and quantity. it helps to know the total amount in every record.

## Features
The project is built trought multiple function such as:
- Register products
- Store product information
- Calculate the total amount
- Generate a summary of all registered products

## Usage
Run the program into main.py
The program will ask you to enter information in product name, price, quantity. After that, it will tell you if you want to record another product or not. At the end, the program will show you a summary of all registered products.
